http://zx.oberon2.ru/forum/viewtopic.php?f=5&t=4

Language: Oberon-2
Development system: XDev/ZXDev, XDev/WinDev

Now just the title is completed.

This project is non-commercial and open source.

Project structure:

  Basic - sources of Dark Woods 2 FreeBASIC port (by Zap)
  Win32 - Dark Woods for Win32 (WinDev)
  ZxSpec - Dark Woods for ZX Spectrum (ZXDev)

To rebuild the game:

1. Download XDev from https://github.com/Oleg-N-Cher/BB-XDev
2. Download DarkWoods from https://github.com/Oleg-N-Cher/DarkWoods

Unpack DarkWoods into XDev folder. Must be at XDev/Games/DarkWoods

Open a module from Win32/Mod (or ZxSpec/Mod) and rebuild/start (F12).

If you want to port the game to a new platform, please contact me:
